# Цветовая палитра

Наша библиотека использует следующие основные цвета:

## Основные цвета

- **Primary:** `#6c5ce7` — основной цвет для кнопок и акцентов.
- **Secondary:** `#f8f9fa` — вторичный цвет для фонов и элементов.

## Фоновые цвета

- **Background:** `#ffffff` — белый фон.
- **Background Disabled:** `#f8f9fa` — фон для неактивных элементов.

## Цвета текста и границ

- **Text:** `#2d3436` — темный текст.
- **Border:** `#dee2e6` — серая граница.
